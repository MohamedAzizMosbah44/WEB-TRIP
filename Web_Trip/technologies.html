Cette partie est didiée à présenter les technologies utilisees pour fabriquer notre site web. Dans cette page, nous avons utilisé des technologies avancées qui ont permis de concevoir et de construire ce site web de manière efficace et performante. Ces outils et langages de programmation nous ont offert la flexibilité nécessaire pour créer une interface conviviale et un design responsive, tout en assurant une expérience utilisateur fluide.

HTML

Langage de base pour créer des pages web. Il permet de structurer le contenu avec des balises.

CSS

Langage utilisé pour styliser les pages web. Il permet de contrôler la mise en page, les couleurs, les polices, etc.

JavaScript

Langage de programmation utilisé pour ajouter de l'interactivité aux sites web.

PHP

Langage de script côté serveur utilisé pour le développement web, permettant de générer du contenu dynamique.

GSAP

Bibliothèque JavaScript pour créer des animations web hautes performances.

WAMPServer

Environnement de développement local pour Windows qui inclut Apache, MySQL et PHP.

Pour plus d'informations sur les languages de programmation, nous vous invitions à en apprendre plus ici.
